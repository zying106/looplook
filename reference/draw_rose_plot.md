# Internal: Draw Rose Chart (Loop Counts)

Visualizes the proportion of loop types based on LOOP COUNTS.

## Usage

``` r
draw_rose_plot(data_df, project_name, filename, color_vec)
```

## Arguments

- data_df:

  Data frame containing loop information.

- project_name:

  Character string for the project title.

- filename:

  Character string for the output filename.

- color_vec:

  Named character vector for colors.

## Value

A ggplot object representing the rose plot.
